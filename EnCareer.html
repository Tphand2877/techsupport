Computer Self-Repair Handbook

Essential laptop "first aid" tips for students before heading to the repair shop.

1. Computer Won't Turn On (No Power)

Don't panic! In many cases, the laptop is just experiencing a static charge buildup or a loose cable.

  • Check Power Supply and Charger Ensure the outlet has power and the charging indicator on the laptop (if available) is lit. Try borrowing a compatible charger to test.
  • Static Discharge (Hard Reset) Unplug the charger and remove the battery (if removable). Press and hold the power button for 30-60 seconds. Plug the charger back in (without the battery) and turn it on.
  • Check Temperature If the laptop was running heavy tasks and shut down suddenly, it might be overheating. Let it rest for 30 minutes in a cool place before restarting.

2. Slow Performance or Freezing (Lag/Hang)

A sluggish computer is often caused by RAM overflow, Full Disk usage, or too many background apps.

  • Check Task Manager (Ctrl + Shift + Esc) Look at the CPU, Memory, and Disk columns. If an app is consuming unusual resources, right-click and select "End Task".
  • Disable Startup Apps Go to the Startup tab in Task Manager, right-click unnecessary apps (like Skype, Zalo, IDM...) and select "Disable".
  • Clean Up Temporary Files Press Win + R, type %temp% and delete all files. Repeat the process with the commands temp and prefetch.

3. Wi-Fi Issues or Unstable Connection

Network connection errors at dorms or campus are extremely common.

  • Forget Network and Reconnect Go to Wi-Fi settings, right-click the problematic network name, and select "Forget". Then re-enter the password to connect.
  • Restart Network Adapter Press Win + X, select Network Connections > Change adapter options. Right-click Wi-Fi, select Disable, wait 10s, then Enable it again.

4. Blue Screen of Death (BSOD)

This is usually caused by software conflicts, driver errors, or hardware issues.

  • Note the Stop Code Look for the capitalized text at the bottom of the blue screen (e.g., MEMORY_MANAGEMENT, CRITICAL_PROCESS_DIED) to Google specific solutions.
  • Uninstall Recently Added Software If the blue screen appeared after installing new software or updating a driver, enter Safe Mode to uninstall them.
Important Note: If the blue screen persists along with beeping sounds, it's likely loose RAM or a failing hard drive. Back up your data immediately and contact TechSupport!
